Compile CHL source files into CHL binary file

Syntax
  chlasm -decompile -st filename -h files -i filename -o path [-a files] [-hl value] [-dc] [-rln]

Arguments
  -st  subtypes table
  -h   C header files or directory
  -i   input CHL file
  -o   output directory
  -a   files for alias definitions
  -hl  heuristic level
  -dc  define unknown constants
  -rln try to respect original line numbers
